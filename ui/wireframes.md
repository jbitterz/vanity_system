# Vanity UI Wireframes (Text)

## 1. Landing / Catalog
- Header: logo left, nav links (Catalog, About, Login, Cart) right.
- Hero filter bar: dropdown (All Brands, Sephora, MAC, Maybelline, Olay, LOréal) + search input + sort selector.
- Product grid: 3 cards per row desktop, 1 per row mobile. Each card: image, brand pill, name, price, rating, `Add to Cart` button.
- Cart preview drawer icon showing item count.

## 2. Product Detail
- Breadcrumb (Home > Brand > Product).
- Left: 400px image carousel; right: name, price, stock badge, quantity stepper, add button, highlights list.
- Tabs: Description, Ingredients, Reviews.

## 3. Cart Page
- Table listing products with thumbnail, name, brand, price, quantity selector, subtotal, remove icon.
- Totals summary card on right: subtotal, shipping selector, tax estimate, discount field, total + Checkout button.
- Continue shopping link.

## 4. Checkout
- Progress indicator (Cart > Shipping > Payment > Review).
- Shipping form: full name, phone, address lines, city, region, postal, country dropdown.
- Payment section: card fields or saved payment tokens.
- Order summary sidebar with editable cart lines and recalculated totals.
- Place Order button + secure badge.

## 5. Login / Signup
- Split card with form on right, brand imagery on left (desktop) and stacked on mobile.
- Inputs: email, password, remember me toggle, forgot password link.
- Signup adds full name + password confirmation.

## 6. Profile / Order History
- Left nav: Profile, Addresses, Orders, Security.
- Orders list: status chips, totals, date. Selecting order reveals modal/panel with items, shipment timeline, download invoice.

## 7. Admin Dashboard
- Metrics cards (Total Sales, Orders Today, Low Stock count).
- Products table with inline edit for stock.
- Order management table with status dropdown per row.
